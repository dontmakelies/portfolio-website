Don't Make Lies Website Design

Fall 2023, I learned basic HTML and CSS to create and design my first website from scratch. Using tutorials online, I designed an interactive homepage, portfolio pages, and an about me page. There are also links to my shop and blog in the site.

The website is a different style from my original site using WordPress to challenge myself to learn something new and see what basic coding was like.

The GitHub repository for the site can be found here.


The main feature of the site is the isometric studio illustration on the homepage. I designed it to be interactive so users can navigate to different pages or links using parts of the room.

The room was created using Adobe Illustrator.

Here are some of the sketches of the studio room and wireframes for the website. I kept most of the same pages from the orginal website (portfolio, about, shop, etc.).

To learn HTML, I watched YouTube tutorials and relied on websites like W3schools and GeeksforGeeks to learn each of the new elements and CSS styles.
